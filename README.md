# Frontend - Sistema de Ventas y Despacho

## Descripción

Aplicación frontend desarrollada para interactuar con los microservicios de Ventas y Despacho mediante una API REST. Permite a los usuarios visualizar y gestionar la información del sistema.

## Tecnologías

- Angular
- TypeScript
- HTML
- CSS
- Bootstrap

## Instalación

```bash
git clone https://github.com/val4nti/Prueba3-devops.git
cd Prueba3-devops
npm install
ng serve
```


## Integración

Este proyecto consume los microservicios de **Ventas** y **Despacho**.
